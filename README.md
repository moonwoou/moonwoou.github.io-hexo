# moonwoou.github.io-hexo

## moonwoou.github.io
https://moonwoou.github.io

## package.json
```
"build": "hexo generate",
"clean": "hexo clean",
"deploy": "hexo deploy --generate",
"server": "hexo server"
```

## Scripts
```
$ npm run build
$ npm run clean
$ npm run deploy
$ npm run server

새 포스트 만들기
$ hexo new post [post_name]
```
